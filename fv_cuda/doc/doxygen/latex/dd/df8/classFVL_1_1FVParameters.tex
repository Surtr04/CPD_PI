\subsection{Detailed Description}
\hyperlink{classParameter}{Parameter} file reader class. This class is used to parse a XML file following the specified format to input parameters to FVLib

XML file structure should be as follows:

$<$?xml version=\char`\"{}1.0\char`\"{} encoding=\char`\"{}UTF-\/8\char`\"{} ?$>$ $<$fvlib$>$ $<$parameters$>$ $<$param1 value=\char`\"{}param1\_\-val\char`\"{} opt\_\-arg=\char`\"{}opt\_\-arg\_\-val\char`\"{}$>$ $<$param2 value=\char`\"{}0.05\char`\"{}$>$ $<$/parameters$>$ $<$/fvlib$>$
